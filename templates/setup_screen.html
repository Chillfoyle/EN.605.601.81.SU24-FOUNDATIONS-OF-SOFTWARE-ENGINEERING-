Let's get started...

Main Menu
Player Input Table
  • Pick a token color: Red, Yellow, Green, or Blue.
  • Click the white text box next to the color you want and type your name.
  • There has to be at least one player, but up to four people can play!
  • Every player must have a different color.
  • Each player's name must be different.
Category Input Table
  • Click each row to select a category from the drop-down list.
  • You must select four categories to play.
